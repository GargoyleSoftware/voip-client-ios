\subsection{\-Detailed \-Description}
\-User registration is controled by \doxyref{\-Linphone\-Proxy\-Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} settings.\par
 \-Each \doxyref{\-Linphone\-Proxy\-Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} object can be configured with registration informations like \doxyref{proxy address }{p.}{group__proxies_ga147377ddae32abf4d3b3777cf0120aed} , \doxyref{user id }{p.}{group__proxies_gaf18076ceb04bc4e779c6e1096e2ef412}, \doxyref{refresh period }{p.}{group__proxies_ga9fe1afeb1c29af86b9322e4e900a05a0}, and so on. \par
 \-A created proxy config using \doxyref{linphone\-\_\-proxy\-\_\-config\-\_\-new()}{p.}{group__proxies_gaaedcf1a592cedba55ad05167abd2d7ce}, once configured, must be added to \doxyref{\-Linphone\-Core}{p.}{group__initializing_gaa5cf635b82dd338e7ee2dd3599d05f0a} using function \doxyref{linphone\-\_\-core\-\_\-add\-\_\-proxy\-\_\-config()}{p.}{group__proxies_gaccdcbca377b771625aeea9c251b597c7}. \par
 \-It is recommended to set a default \doxyref{proxy config }{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} using function \doxyref{linphone\-\_\-core\-\_\-set\-\_\-default\-\_\-proxy()}{p.}{group__proxies_ga466cfc7b5f7a63e1efb77732ac3b6de2}. \-Once done, if \doxyref{a proxy config }{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} has been configured with attribute \doxyref{enable register }{p.}{group__proxies_gadca1a0fe47bceff0e58d7057672519b6} , next call to \doxyref{linphone\-\_\-core\-\_\-iterate()}{p.}{group__initializing_ga7bdac8ac386483fc4e0876e4c5d13755} triggers a \-S\-I\-P register. \par
 \-Registration status is reported by \doxyref{\-Linphone\-Registration\-State\-Cb}{p.}{group__initializing_gae2118ee5fdce307bd608a2d40bfa3829}. \par
 \par
 \-This pseudo code demonstrates basic registration operations\-: \par

        /*create proxy config*/
        proxy_cfg = linphone_proxy_config_new();
        /*parse identity*/

        LinphoneAuthInfo *info;
        if (password!=NULL){
                info=linphone_auth_info_new(linphone_address_get_username(from)
      ,NULL,"secret",NULL,NULL); /*create authentication structure from identity*/
                linphone_core_add_auth_info(lc,info); /*add authentication info
       to LinphoneCore*/
        }       
        // configure proxy entries
        linphone_proxy_config_set_identity(proxy_cfg,identity); /*set identity
       with user name and domain*/
        const char* server_addr = linphone_address_get_domain(from); /*extract
       domain address from identity*/
        linphone_proxy_config_set_server_addr(proxy_cfg,server_addr); /* we
       assume domain = proxy server address*/
        linphone_proxy_config_enable_register(proxy_cfg,TRUE); /*activate
       registration for this proxy config*/
        linphone_address_destroy(from); /*release resource*/

{\bfseries \-Authentication\-:} \par
\-Most of the time, registration requires \doxyref{authentication}{p.}{group__authentication} to succed. \doxyref{\-Linphone\-Auth\-Info}{p.}{group__authentication_ga0f8289c1aab407992864a68869107f4f} info must be either added to \doxyref{\-Linphone\-Core}{p.}{group__initializing_gaa5cf635b82dd338e7ee2dd3599d05f0a} using function \doxyref{linphone\-\_\-core\-\_\-add\-\_\-auth\-\_\-info()}{p.}{group__authentication_ga62ac5bb34f0342f7edc733c42473b7aa} before \doxyref{\-Linphone\-Proxy\-Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} is added to \-Linphone core, or on demand from call back \doxyref{\-Auth\-Info\-Requested}{p.}{group__initializing_ga2673e7baaa3ddec6190902fc76204b29} . \par
 \par
{\bfseries \-Unregistration\-:} \par
 \-Unregistration or any changes to \doxyref{\-Linphone\-Proxy\-Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} must be first started by a call to function \doxyref{linphone\-\_\-proxy\-\_\-config\-\_\-edit()}{p.}{group__proxies_ga37cac414aba9e8e4135c1f208c0175f8} and validated by function \doxyref{linphone\-\_\-proxy\-\_\-config\-\_\-done()}{p.}{group__proxies_ga9b59d963af64ee243dfa4643a85c0cdc} \par
