\subsubsection[{\-Linphone\-Address}]{\setlength{\rightskip}{0pt plus 5cm}{\bf \-Linphone\-Address}}\label{group__linphone__address_gaa017e66840f5ee578168bd469b6c3e91}
\-Object that represents a \-S\-I\-P address.

\-The \-Linphone\-Address is an opaque object to represents \-S\-I\-P addresses, ie the content of \-S\-I\-P's 'from' and 'to' headers. \-A \-S\-I\-P address is made of display name, username, domain name, port, and various uri headers (such as tags). \-It looks like '\-Alice $<$sip\-:{\tt alice@example.\-net}$>$'. \-The \-Linphone\-Address has methods to extract and manipulate all parts of the address. \-When some part of the address (for example the username) is empty, the accessor methods return \-N\-U\-L\-L.
