\subsubsection[{\-Lp\-Config}]{\setlength{\rightskip}{0pt plus 5cm}typedef struct \-\_\-\-Lp\-Config {\bf \-Lp\-Config}}\label{group__misc_ga047b6550aab7dd244268e695fab98fd6}
\-The \-Lp\-Config object is used to manipulate a configuration file.

\-The format of the configuration file is a .ini like format\-:
\begin{DoxyItemize}
\item sections are defined in []
\item each section contains a sequence of key=value pairs.
\end{DoxyItemize}

\-Example\-: 
\begin{DoxyCode}
 [sound]
 echocanceler=1
 playback_dev=ALSA: Default device

 [video]
 enabled=1
\end{DoxyCode}
