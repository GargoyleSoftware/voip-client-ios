\subsection{\-Detailed \-Description}
{\bfseries  \-Exchanging text messages} \par
 \-Messages are sent using \doxyref{\-Linphone\-Chat\-Room}{p.}{group__chatroom_ga37902f3fc715b7f6451c241d0efd0299} object. \-First step is to create a \doxyref{chat room }{p.}{group__chatroom_ga02c675b2181e77397694e801cd606e85} from a peer sip uri. 
\begin{DoxyCode}
LinphoneChatRoom* chat_room = linphone_core_create_chat_room(lc,"
      sip:joe@sip.linphone.org");
\end{DoxyCode}


\par
\-Once created, messages are sent using function \doxyref{linphone\-\_\-chat\-\_\-room\-\_\-send\-\_\-message()}{p.}{group__chatroom_ga5d70a868f932e0793991c8aca1289591} . 
\begin{DoxyCode}
linphone_chat_room_send_message(chat_room,"Hello world"); /*sending message*/
\end{DoxyCode}
 \par
\-Incoming message are received from call back \doxyref{\-Linphone\-Core\-V\-Table.\-text\-\_\-received}{p.}{struct__LinphoneVTable_aa2d467de0d6fcf1ca5c74b55b14da44e} 
\begin{DoxyCode}
void text_received(LinphoneCore *lc, LinphoneChatRoom *room, const 
      LinphoneAddress *from, const char *message) {
        printf(" Message [%s] received from [%s] \n",message,
      linphone_address_as_string (from));
}
\end{DoxyCode}
