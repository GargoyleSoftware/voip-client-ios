\subsection{\-Detailed \-Description}
{\bfseries \-Buddies and buddy list} \par
\-Each buddy is represented by a \doxyref{\-Linphone\-Friend}{p.}{group__buddy__list_ga2f0508bab6d8ded431ce8b1a679a1522} object created by function \doxyref{linphone\-\_\-friend\-\_\-new()}{p.}{group__buddy__list_ga08e7cb027cf8c90e0c1bcab39f8199a5}. \-Buddy configuration parameters like \doxyref{sip uri }{p.}{group__buddy__list_ga61903a0cb9927f100e7137c05057d04d} or \doxyref{status publication }{p.}{group__buddy__list_ga9ab0c9222edcc0a8e25fb068aee88bc0} policy for this \doxyref{friend }{p.}{group__buddy__list_ga2f0508bab6d8ded431ce8b1a679a1522} are configurable for each buddy. \par
\-Here under a typical buddy creation\-: \par
 
\begin{DoxyCode}
LinphoneFriend* my_friend=linphone_friend_new_with_addr("
      sip:joe@sip.linphone.org"); /*creates friend object for buddy joe*/
linphone_friend_enable_subscribes(my_friend,TRUE); /*configure this friend to
       emit SUBSCRIBE message after being added to LinphoneCore*/
linphone_friend_set_inc_subscribe_policy(my_friend,LinphoneSPAccept); /* accept
       Incoming subscription request for this friend*/
\end{DoxyCode}
 \doxyref{friends }{p.}{group__buddy__list_ga2f0508bab6d8ded431ce8b1a679a1522} status changes are reported by callback \doxyref{\-Linphone\-Core\-V\-Table.\-notify\-\_\-presence\-\_\-recv}{p.}{struct__LinphoneVTable_afedfc364c5b5193f10473cd47524400f} 
\begin{DoxyCode}
static void notify_presence_recv_updated (struct _LinphoneCore *lc,  
      LinphoneFriend *friend) {
        const LinphoneAddress* friend_address = linphone_friend_get_address(
      friend);
        printf("New state state [%s] for user id [%s] \n"
                                ,linphone_online_status_to_string(
      linphone_friend_get_status(friend))
                                ,linphone_address_as_string (friend_address));
}
\end{DoxyCode}
 \par
\-Once created a buddy can be added to the buddy list using function \doxyref{linphone\-\_\-core\-\_\-add\-\_\-friend()}{p.}{group__buddy__list_ga3a8e9184320402082bf35226ae8d25b8} . \-Added friends will be notified about \doxyref{local status changes }{p.}{group__buddy__list_gadae69527e9b36e0f29bd80c30275874f} \par
 \-Any subsequente modifications to \doxyref{\-Linphone\-Friend}{p.}{group__buddy__list_ga2f0508bab6d8ded431ce8b1a679a1522} must be first started by a call to function \doxyref{linphone\-\_\-friend\-\_\-edit()}{p.}{group__buddy__list_ga254b01923c79947a5d23981d47639f40} and validated by function \doxyref{linphone\-\_\-friend\-\_\-done()}{p.}{group__buddy__list_ga6e28513bba28f948ee61341384a36958} 
\begin{DoxyCode}
linphone_friend_edit(my_friend); /* start editing friend */
linphone_friend_enable_subscribes(my_friend,FALSE); /*disable subscription for
       this friend*/
linphone_friend_done(my_friend); /*commit changes triggering an UNSUBSCRIBE
       message*/
\end{DoxyCode}


{\bfseries  \-Publishing presence status } \par
\-Local presence status can be changed using function \doxyref{linphone\-\_\-core\-\_\-set\-\_\-presence\-\_\-info()}{p.}{group__buddy__list_gadae69527e9b36e0f29bd80c30275874f} .\-New status is propagated to all friends \doxyref{previously added }{p.}{group__buddy__list_ga3a8e9184320402082bf35226ae8d25b8} to \doxyref{\-Linphone\-Core}{p.}{group__initializing_gaa5cf635b82dd338e7ee2dd3599d05f0a}.

{\bfseries \-Handling incoming subscription request} \par
 \-New incoming subscription requests are process according to \doxyref{the incoming subscription policy state }{p.}{group__buddy__list_ga9ab0c9222edcc0a8e25fb068aee88bc0} for subscription initiated by \doxyref{members of the buddy list. }{p.}{group__buddy__list_ga3a8e9184320402082bf35226ae8d25b8} \par
 \-For incoming request comming from an unknown buddy, the call back \doxyref{\-Linphone\-Core\-V\-Table.\-new\-\_\-subscription\-\_\-request}{p.}{struct__LinphoneVTable_ae66dffa2d5e6b3be881552cab3024ec2} is invoked.

\par
 \-A complete tutorial can be found at \-: \doxyref{\-Registration tutorial}{p.}{group__buddy__tutorials}
